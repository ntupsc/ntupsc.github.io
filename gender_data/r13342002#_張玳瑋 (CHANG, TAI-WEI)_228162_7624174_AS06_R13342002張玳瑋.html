建立性別比例指標

# 整理為 wide format
births_wide <- births %>%
  pivot_wider(names_from = 性別, values_from = 人數) %>%
  rename(男性 = `男`, 女性 = `女`) %>%
  mutate(
    性別比 = (女性 - 男性) / (女性 + 男性)
  )

# 取最新年度資料做地圖
latest_year <- max(births_wide$年份)
births_latest <- births_wide %>%
  filter(年份 == latest_year)

🗺 Finding

112 年台灣各縣市新生兒性別比顯示出明顯地區落差，部分中南部縣市如彰化、南投、雲林等地呈現男性明顯多於女性的現象，可能與地方傳統性別偏好有關；相對而言，臺北市、新竹市等都市區性別比例較為平衡。整體而言，多數縣市性別比略偏向男性，反映出生性別結構受到文化與社會因素影響，顯示台灣在性別平等上仍有潛在挑戰，特別是在生育選擇上。

🗺 臺北市 2013–2023 新生兒性別比Finding

從圖中可見，臺北市2013年至2023年新生兒性別比（(女−男)/(女+男)）長期呈現負值，表示男嬰數量持續多於女嬰。儘管性別比波動範圍不大，但始終未突破零軸，最高約為-0.02，最低接近-0.05，顯示男嬰在出生人口中占比仍偏高。這樣的現象可能反映出潛在的社會文化因素，例如家庭對男嬰的偏好，或是自然性別比以外的結構性偏差。值得注意的是，雖然臺北市作為首都，教育程度與性別意識相對普及，性別比仍顯示出長期且穩定的偏差，說明即使在都市地區，性別平等的實踐仍面臨隱性挑戰，值得政策進一步關注。
